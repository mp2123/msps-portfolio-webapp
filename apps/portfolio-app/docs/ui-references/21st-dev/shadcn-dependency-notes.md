# shadcn dependency notes

Several of the archived inspirations assumed:
- shadcn-style structure
- Tailwind CSS
- TypeScript
- default component path conventions similar to `/components/ui`

This repo already meets the core requirements:
- Next.js app with TypeScript
- Tailwind configured
- shadcn-compatible component structure in `src/components/ui`

Referenced shadcn-style primitives already exist in the production app:
- `src/components/ui/badge.tsx`
- `src/components/ui/button.tsx`
- `src/components/ui/card.tsx`
- `src/components/ui/dialog.tsx`
- `src/components/ui/scroll-area.tsx`
- `src/components/ui/command.tsx`
