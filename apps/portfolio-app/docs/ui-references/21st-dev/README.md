# 21st-dev UI Reference Archive

These files capture the user-provided inspiration components from the March 26, 2026 Codex thread.

Purpose:
- preserve the design references one by one
- keep raw inspiration separate from production-safe portfolio components
- document the requested dependencies and intended usage

Important:
- these are reference notes, not drop-in production approvals
- several of the originals are visually strong but expensive, brittle, or overly global in behavior
- the shipped portfolio adapts the ideas instead of blindly mounting every raw component
