# Mixology App UI Reference Archive

These files capture standout interface patterns from the `mixology-app` in this repo.

Purpose:
- preserve strong internal references from sibling apps
- separate raw inspiration from production-safe portfolio components
- document where the pattern came from and what behavior is worth reusing

Important:
- these are reference notes, not approvals to mount the original component unchanged
- the mixology app is optimized for immersion and browsing
- the portfolio needs recruiter-speed clarity, so any reuse should be tighter and more purposeful

Archived reference files in this folder:
- `card-scanner-discovery.md`

Adapted production implementation:
- `src/components/portfolio/sections/artifact-scanner.tsx`

Archive conventions:
- each file records the original source path
- each file summarizes the visual and interaction mechanics
- each file notes portfolio-specific opportunities
- each file ends with adaptation guidance so the idea can be reused without importing the full original experience
