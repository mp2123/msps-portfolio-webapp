# Incoming Drop Zone

Use this folder if you want to dump files in quickly without sorting them first.

Anything placed here is local intake material, not GitHub-ready and not website-ready by default.

Workflow:
- put unsorted project files here
- I will review them, classify them, and move them into the right project folders under `source-material`
- if a new project or theme appears, I will create a new intake bucket for it

Good things to drop here:
- decks and PDFs
- screenshots and exports
- notebooks, queries, or code snippets
- recommendation docs or notes
- videos or Loom exports
- anything that explains the business problem, your role, tools, or measurable outcome
